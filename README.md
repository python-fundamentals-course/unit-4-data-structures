# Unit 4: Data Structures

> **Python Fundamentals**

Choosing between list, dict, tuple, and set — and the list-of-dicts pattern for real data.

## New here?

If this is your first time opening a unit repo, start with
[**GETTING_STARTED.md**](GETTING_STARTED.md) — it covers opening this
Codespace, using Jupyter notebooks, doing the exercises, and checking your
work. You don't need to install anything.

## Learning objectives

By the end of this unit you will be able to:

- Choose the right data structure for a given task
- Use the list-of-dicts pattern to represent tabular data
- Apply list comprehensions and dict operations to process data
